# Bug Bounty Watchlist

Date: 2026-03-31

## Best Immediate Platforms

- `Hacker101 / HackerOne`
  - Why: free training, CTF practice, and a path toward private-program invitations
  - Source: https://www.hackerone.com/hackers/hacker101
- `Bugcrowd`
  - Why: active hacker platform with public-start resources and a large program base
  - Source: https://www.bugcrowd.com/hackers/
- `Apple Security Bounty`
  - Why: official Apple reward path for eligible research
  - Source: https://support.apple.com/en-us/102549
- `Microsoft Security Response Center`
  - Why: active bounty program portfolio and formal report intake
  - Source: https://msrc.microsoft.com/

## First Focus Areas

- Web application auth flaws
- Access control and IDOR
- Stored and reflected XSS
- Sensitive data exposure
- Business logic flaws in officially in-scope targets

## Avoid Early

- Broad active scanning without clear scope
- Any target that requires special approval or invite unless that invite exists
- Anything outside published rules or rate limits
