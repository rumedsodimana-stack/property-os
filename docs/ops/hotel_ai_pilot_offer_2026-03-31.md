# Hotel AI Pilot Offer

Date: 2026-03-31

## Offer

Launch a 14-day AI Reservations + Concierge Pilot for one property or one small hotel group.

## Who It Is For

- Independent hotels
- Boutique and lifestyle hotels
- Small hotel management groups
- Resorts with heavy guest messaging and after-hours call volume

## Outcome We Sell

- Capture reservation intent after hours instead of losing it
- Answer repeat guest questions without front desk overload
- Route guest requests into real operations tasks
- Offer paid upsells like early check-in, room upgrades, late checkout, parking, breakfast, and spa
- Show management a weekly revenue and workload impact report

## What Gets Deployed

### Guest-facing agent

- Web or messaging concierge
- FAQ and property knowledge handling
- Service request intake
- Upsell prompts

### Ops routing

- Housekeeping task creation
- Maintenance task creation
- Front desk trace creation
- Food and beverage request handoff

### Revenue layer

- Offer logic for common upsells
- Payment capture path for paid add-ons
- Basic reporting on requests, leads, and upsells

## Why We Can Deliver Fast

This repo already includes the core parts needed for a believable hotel pilot:

- Multi-agent foundation in `services/intelligence/agentService.ts`
- Guest request routing in `services/operations/guestActionService.ts`
- Payment intent and capture flows in `services/payments/paymentService.ts`
- OTA integration hooks in `services/integrations/otaService.ts`
