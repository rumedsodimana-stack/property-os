# Earnings Notes

Date: 2026-03-31

Purpose: persistent local record of money-related progress so future check-ins can answer "how much did you earn?"

## Tracking Rules

- `amount` is total contract or bounty value
- `received_amount` is cash actually received
- use `status` values like `lead`, `proposal_sent`, `won`, `invoiced`, `paid`, `lost`
- every real deal, proposal, bounty claim, or paid partner referral should be added to `docs/ops/earnings_ledger.csv`
