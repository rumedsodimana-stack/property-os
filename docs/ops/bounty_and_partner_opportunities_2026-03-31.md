## Practical Strategy

### Bounties

- check Cal.com and Algora first
- avoid issues already clearly owned or reserved
- look for low-complexity front-end or TypeScript tasks that can be finished in one session

### Bug bounty

- do not rely on this for April cash
- only use it as optional upside
- focus on one program at a time if pursued

### Partners

- apply for Asksuite referral and reseller tracks
- position ourselves for Retell-based implementation work and white-label deployment

## Revenue Expectation

- public code bounties: small but fast, USD 20 to 500 each in the examples above
- security bounty: high upside, low predictability
- partner channels: indirect but can feed larger service deals
